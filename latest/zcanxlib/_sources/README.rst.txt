.. _zcanxlib:

ZCAN Xlib
#########

zcanxlib is a repository that contains RTOS-independent libraries and modules to be used with ZCAN SoCs.


Each library and module in this repository supports a different set of SoCs from ZCAN.
Refer to their respective documentation for more information.
