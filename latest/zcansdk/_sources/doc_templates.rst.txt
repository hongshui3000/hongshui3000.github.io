.. _doc_templates:

Documentation templates
########################

The |ZSDK| documentation provides templates for documenting sample applications and libraries.

Use these templates to start documenting your application or library.

You can find the templates in ``ZCAN-IOT/ZCAN-sdk/doc/zcansdk/templates``. This folder also contains an RST "cheat sheet" to help you getting started with RST.
