Building and Running
********************

This application can be built and executed on oc rcc as follows:

.. zephyr-app-commands::
   :zephyr-app: samples/rcc
   :host-os: windows
   :board: ocean_evb
   :goals: run
   :compact:

To build for another board, change "ocean_evb" above to that board's name.
