.. _release_notes:

Release notes
#############

See the release notes for information about specific |ZSDK| releases.

This page is included only in the latest documentation, because it might contain issues that are discovered after a release is tagged.

.. note::
   A "99" at the end of the version number of this documentation indicates continuous updates on the master branch since the previous major.minor release.
   When looking at this latest documentation, be aware of the following aspects:

   * The release note pages that are available in the latest documentation might differ slightly from the release notes that were included in the respective |ZSDK| release at its release date.
     Therefore, to see the official version of the release notes for a specific |ZSDK| release, switch to the documentation for the corresponding |ZSDK| version using the selector in the upper left-hand corner.
